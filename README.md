# 特别注意:
-1 **超级重点**:如果解冻有异常,窗体不显示等, 请在游戏设置中调节成"无边框全屏", 能窗口化尽量窗口化!(无异常随意.)
## 0: 很多用户反馈更新1.40后导致ALT+左键失灵, 原因是开启"妙控拖拽"功能后,使得按住ALT键会屏蔽左键与中键原功能!与原神等需要ALT点击界面的操作冲突! 请自行根据情况开关妙控拖拽功能)

1:对于游戏窗体出现奇奇怪怪的问题,建议勾选上:仅最小化.

2.有些程序是能冻结,但反调试引擎也会在程序未响应超时后强制崩溃退出,可以用自动呼吸来解决这个问题, 所以雪藏除解决冻结问题外,还要考虑对抗反调试反作弊引擎.
目前稳定冻结的热门网游,包含但不仅限于:LOL,永劫无间,尘白禁区,无畏契约,原神,崩铁,崩坏3(米哈游全攻略了)cs:go,我的世界MC, APEX,等等等等

3.网游可保证不触发反作弊等封号,因为无修改数据操作,这个有大量玩家已实测,可放心食用, 如果用雪藏HsFreezer都封号, 那么你休眠待机也会封, 这么说明白吧?

4.V2ray导致闪退的用户, 可重新安装.NET Runtime，把v2rayN数据清空重新安装v2ray即可

5.多设备玩steam游戏,由于云存档机制问题, 要注意有没有自动云存档，雪藏中的游戏,没退出前steam可能会没有把游戏存档即时上传更新, 换设备后, 会造成存档回滚. (感谢:古赫分享)
(单设备就尽情的造,设备太多也是一种烦恼哈哈)

6.有些朋友在某xxxx520下载的学习版单机, 冻结解冻后偶尔提示可能会提示致命错误, 这时候,你别点确定, 你照常玩确实没事,但你一点确定按钮就崩了,请悉知.

7.经常休眠的朋友,请保证C盘有大于物理内存空间的硬盘空间,否则会引擎未知错误.如果这样还崩溃, 还有一个小技巧就是:休眠/睡眠之前,把他放出来再休眠/睡眠试试.
(卡普空部分游戏:如生化危机,怪猎崛起等,因为AMD驱动与DX12有某些冲突,可能休眠会引起崩溃,当然,这个驱动版本是玄学,可自行测试,现阶段,DX11更为稳定,特别是休眠唤醒时!)

8.不要手贱去冻结有关鼠标光标手势加强类软件:如quicker,wg2,lily等,别问为什么.(光标卡顿就是因为这个,补救放案就是快捷键解冻)

9.一定要预留一定量的物理内存与显存来冻结游戏(特别是独显),因为你再怎么塞你都看不到内存占满,其实内存达到80%以上已经各种在与虚拟内存交换了, 不要塞得满满的,否则有一些系统级调度问题,比如游戏崩溃,或者解冻时出现一段时间卡顿才出现(这就是从虚拟内存导出的表现,请须知).

10.如老头环,APEX等有小蓝熊加持的游戏,也许能冻结成功,但这类引擎检测到进程暂停一段时间后可能会强行结束进程, 建议用自呼吸模式尝试.
